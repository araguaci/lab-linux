## 🚀 Especificações

### Hardware Recomendado
- **Raspberry Pi 4B** - Recomendado (4GB+ RAM)
- **Raspberry Pi 3B+** - Compatível com limitações
- **Alimentação** - 5V 3A+ (USB-C)
- **Cartão SD** - 32GB+, Classe 10+

### Sistemas Operacionais
- Raspberry Pi OS (Debian)
- Ubuntu 22.04 LTS
- Sistemas específicos (FreePBX, Kali, etc)
