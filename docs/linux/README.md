## 🔧 Ferramentas Recomendadas

### Utilitários Essenciais
```
ncdu - Analisador de uso de disco
ripgrep - Busca de texto muito rápida
fzf - Fuzzy finder interativo
zoxide - Navegação inteligente
exa - Versão moderna de 'ls'
```

### Ferramentas de Rede
```
mtr - Traceroute avançado
dnsutils - Utilidades DNS
tcpdump - Captura de pacotes
wireshark - Análise de rede
```

### Monitoramento
```
htop - Monitor de processos interativo
iotop - Monitor de I/O
glances - Monitor de sistema completo
```
