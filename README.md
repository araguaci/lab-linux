# Lab Linux 🐧

Documentação completa sobre Linux, Docker, WSL, infraestrutura e muito mais.

## 📚 Estrutura da Documentação

A documentação foi reorganizada por tópicos para melhor navegação:

### [📖 Overview](docs/overview/)
Visão geral do projeto e documentação principal.

### [🐧 Linux](docs/linux/)
Comandos essenciais, ferramentas, instalação e administração de sistemas Linux.

### [🐳 Docker](docs/docker/)
Tudo sobre containers: comandos, integração com WSL, networking e mais.

### [🪟 WSL](docs/wsl/)
Windows Subsystem for Linux: instalação de distros, GUI, systemd e customização.

### [🍓 Raspberry Pi](docs/raspberry-pi/)
Guias para Raspberry Pi: rede, GUI, imagem do sistema e aplicações.

### [🌐 Networking](docs/networking/)
Configuração de rede: Nginx, NetworkManager, DNS, Proxmox e infraestrutura.

### [📊 Monitoring](docs/monitoring/)
Ferramentas de monitoramento: Zabbix, diagnóstico de CPU, análise de tráfego HTTP.

### [☎️ VoIP & PBX](docs/voip/)
Telefonia IP: Asterisk, FreePBX, Issabel e sistemas de PBX.

### [🔐 Security](docs/security/)
Segurança: permissões, sudo, privilégios e boas práticas.

### [📎 References](docs/references/)
Referências e documentação externa.
